# Notas sobre HTML

## Lang
- `lang` = linguagem do site

## Head
Tudo que está dentro do `<head>` não será exibido para o usuário.

## Meta
- `meta` = metatags = metadados (são lidas apenas pelos navegadores)

### Atributos Meta
- `charset` = sistema de caracteres
- `viewport` = área visível do usuário na página
- `description` = descrição do site
- `keywords` = palavras-chave do site
- `author` = criador do site

## Cabeçalhos
Tags de títulos `h1` ao `h6`:
- `h1` usado para o conteúdo principal
- As demais (`h2` a `h6`) usadas para subconteúdos dentro do conteúdo principal
- Ao digitar `tag*número` criamos a quantidades de tags pelo número multiplicado automaticamente

## SEO
Search Engine Optimization [otimização para motores de busca]

## tags de formatação de texto
- `<strong>` = texto em negrito (importante semanticamente, enfáse)
- `<b>` = texto em negrito (apenas visual)
- `<em>` = texto em itálico (ênfase semântica)
- `<i>` = texto em itálico (apenas visual)
- `<mark>` = texto marcado/destacado
- `<small>` = texto menor
- `<del>` = texto deletado/riscado
- `<ins>` = texto inserido/sublinhado
- `<sub>` = texto subscrito
- `<sup>` = texto sobrescrito 
- `<hr>` = linhas entre seções
- `<u>` = texto sublinhado
- `<del>` = texto tachado
- `<sup>` = texto sobrescrito
- `<sub>` = texto subrescrito
